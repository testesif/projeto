SoloFertil

Utilizamos Arduino, Python e sensores para detectar quando o solo está apto para plantio.

Como Funciona?

Nosso sistema coleta dados do solo usando sensores conectados a um Arduino. Esses dados são processados em Python para determinar se o solo está em condições ideais para o plantio.

  • Sensor de umidade do solo.
  • Sensor de temperatura.
  • Análise de pH.
